API简介
1、API的概念
（Application Programming Interface，应用程序编程接口）
是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码或内部工作的细节。

2、API的特点：
（1）API是一个明确定义的接口，可以为其它软件提供特定服务
（2）API可以小到只包含一个单独的函数，也可以大到包含数以百计的类、方法、全局函数、数据类型、枚举类型和常量等。
（3）API的实现可以是私有的，也可以是开源的

3、API的分类
（1）面向对象语言的API；例：java API
（2）库与框架的API；例：Windows API、Windows DirectX
（3）API与协议；例：LDAP应用程序接口
（4）API与设备接口；例：PC BIOS调用接口、ASPI for SCSI设备接口
（5）Web API；例：Google地图API、新浪微博API、阿里云API市场

4、为什么要使用API
（1）快速扩展功能
（2）避免“造轮子”，提高开发效率
（3）降低模块之间的耦合度
